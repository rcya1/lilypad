Multiple Hypothesis Testing

  • A scientist conducted a series of tests on brain voxels of a dead salmon
    • They did 8000 tests with α=0.001\alpha = 0.001α=0.001
    • They rejected 8 null hypothesis, i.e. found that 8 voxels were active in a dead fish
  • The reason was because with α=0.0001\alpha = 0.0001α=0.0001 and 8000 tests, you are expected to find 888 rejections
  • In general, when you conduct multiple hypothesis tests, it is inevitable you will reject some of them

Bonferroni Correction

  • Instead of making each test's type I error less than α\alphaα, we set it up so:
    • The overall type I error is less than α\alphaα
    • This can be done by rejecting if less than α/m\alpha / mα/m, where mmm is the number of tests
    • Can use the union bound to show that this makes the overall type I error probability less than α\alphaα
  • This is incredibly conservative

Benjamini-Hochberg Method

  • Bonferonni correction attempts to make the expected total number of false positives less than α\alphaα, which was very conservative
  • Instead, another goal might be to attempt to make the proportion of false positives out of all positives is less than α\alphaα
  • We can accomplish this by:
    • Ordering all ppp values p1,p2,…,pmp_1, p_2, \dots, p_mp1​,p2​,…,pm​
    • Let imax⁡i_{\max}imax​ be the largest iii such that pi≤α/m∗ip_i \leq \alpha / m * ipi​≤α/m∗i
    • Reject all pip_ipi​ such that i≤imax⁡i \leq i_{\max}i≤imax​
  • This works if the test statistics are all independent from each other
  • This guarantees the rejection region to be somewhere between Bonferonni and typical hypothesis testing
