Hypothesis Test

A hypothesis test takes the form
H0:θ∈Θ0H_0 : \theta \in \Theta_0H0​:θ∈Θ0​ vs H1:θ∈Θ1H_1 : \theta \in \Theta_1H1​:θ∈Θ1​

  • We typically characterize H0H_0H0​ as the null hypothesis, or the status quo
  • H1H_1H1​ is a revelation or something that you prove

Error Types

Type I Error

A Type I error is when H0H_0H0​ is true, but the test concludes H1H_1H1​ is true. This is considered more serious in things like a criminal trial.

Type II Error

A Type II error is when H1H_1H1​ is true, but the test concludes H0H_0H0​ is true.

The probability of a test committing an error depends on the true value of the parameter
