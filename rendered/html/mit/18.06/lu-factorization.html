LU Factorization

Writing the matrices that bring a matric AAA to REF as G1G_1G1​, G2G_2G2​, ⋯\cdots⋯, GiG_iGi​, we can write:

  • L=(Gi⋯G2G1)−1L = (G_i\cdots G_2G_1)^{-1}L=(Gi​⋯G2​G1​)−1
  • U=Gi⋯G2G1AU = G_i \cdots G_2G_1AU=Gi​⋯G2​G1​A

UUU is the REF of AAA and LLL is lower triangular iff there are no row swaps / permutations done in any of the GiG_iGi​.

Solving Systems of Linear Equations

Useful when AAA is invertible and you need to solve Ax=bAx = bAx=b because substition allows you to easily solve equations for triangular matrices

  • Solve Lc=bLc = bLc=b for ccc
  • Then solve Ux=cUx = cUx=c

LDU Factorization

LLL will always have 1s for pivots but UUU may not

  • Can solve for this by introducing a diagonal matrix DDD that normalizes UUU
