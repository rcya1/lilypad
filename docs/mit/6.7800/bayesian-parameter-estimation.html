Bayes Risk for Estimation

We use x^(y)\vec{\hat{x}}(\vec{y})x^(y) to denote our estimate of x\vec{x}x based on our observation y\vec{y}y

We need some "measure of goodness" to denote how good this estimator is

  • We choose some scalar valued cost function C(a,a^)C(\vec{a}, \vec{\hat{a}})C(a,a^)
  • We want our estimator to minimize the expected value of this, which is known as the Bayes risk
    E[C(x,f(y))]\exp{C(\vec{x}, f(\vec{y}))}E[C(x,f(y))]
    • This expectation is jointly over both x\vec{x}x and y\vec{y}y
  • Similar to the Bayes hypothesis testing case, this can be solved for by minimizing for each different value y\vec{y}y separately
    • Our goal is then to, for each y\vec{y}y, find the a\vec{a}a that minimizes
      ∫−∞∞C(x,a)px∣y(x∣y)dx\int_{-\infty}^\infty C(\vec{x}, \vec{a}) p_{\vec{x} | \vec{y}}(\vec{x} | \vec{y}) d\vec{x}∫−∞∞​C(x,a)px∣y​(x∣y)dx

When we can break down C(x,a)=∑iCi(xi,ai)C(\vec{x}, \vec{a}) = \sum_i C_i(x_i, a_i)C(x,a)=∑i​Ci​(xi​,ai​), then we can solve for each component separately

  • For now, we will focus on just the scalar case for various cost functions

Minimum Absolute Error (MAE)

In MAE, the cost function is C(a,a^)=∣a−a^∣C(a, \hat{a}) = |a - \hat{a}|C(a,a^)=∣a−a^∣

Claim

The MAE estimate is the median of the posterior belief, that is, the value of xxx that that cdf(x)=1/2\text{cdf}(x) = 1/2cdf(x)=1/2

Sketch:

  • Let the estimate be aaa
  • We can break the absolute value into x−ax - ax−a and a−xa - xa−x and break the expression into two integrals with one going from −∞-\infty−∞ up to aaa and the other going from aaa to ∞\infty∞
  • Differentiating and setting to 0 yields that aaa must be positioned such that the densities on both sides are equal

We can generalize this to vectors via the L1L^1L1 norm, which is componentwise absolute values

The BLS estimator is unbiased

BLS Orthogonality

An estimator x^\hat{x}x^ is the BLS estimator iff the estimation error is orthogonal to any vector-valued function ggg of the data:
E[(x^(y)−x)g(y)T]=0\exp{(\hat{x}(y) - x)g(y)^T} = 0E[(x^(y)−x)g(y)T]=0

The idea is the error should be uncorrelated with any function of the data we construct

  • Therefore, we cannot transform the data to further reduce the error in our estimate
