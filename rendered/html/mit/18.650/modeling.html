Modeling

Suppose we had a linear regression model and we just added spurious variables that are essentially just noise

  • If we compute the least squares regression model using the spurious models versus without, we will find a better fit
  • One interpretation of this is that we are essentially projecting onto a higher subspace by increasing the number of features
    • Remember that an intuitive interpretation of linear regression is projecting a nnn dimensional YYY onto the kkk dimensional column space of XXX
    • The codimension of the column space of XXX is therefore n−kn - kn−k
      • Codimension of a subspace = dimension of main subspace - dimension of subspace
    • Increasing the dimension of XXX therefore decreases the codimension and leaves less variables in the residual, which exists in a subspace that has dimension equal to the codimension
    • Therefore, the residual length will naturally decrease
  • In fact, if we add enough spurious variables to make the dimension of XXX equal to nnn, then we will get a perfect fit
  • This is a phenomenon known as overfitting

A lot of times in the real world, we have a ton of variables, many of which are junk and we need to remove some of them

  • This is known as model selection
  • One way to do this is to do a test for βj∗≠0\beta_j^* \neq 0βj∗​=0 for each of the variables (Wald's test)
    • However we are doing multiple hypothesis tests and therefore need some kind of correction
      • Bonferonni correction is too conservative and BH requires the test statistics to be independent, which is not true here
    • This also only really works in the asymptotics and we are dealing with an effective sample size of n−kn - kn−k
      • In some cases, we might have n<kn < kn<k
  • In reality, what we really want to do is test if βj∗=0\beta_j^* = 0βj∗​=0 (kind of like the inverse of Wald's test)
    • Our null hypothesis is that it is ≠0\neq 0=0
    • But our current hypothesis testing framework doesn't allow us to do this

R Squared

We want to assign a score to each subset SSS of the variables that we choose

  • We can't just choose the residual length because that would just be all of the variables
  • We want a dimensionless value between 000 and 111
R Squared

The R2R^2R2 or coefficient of determination measures the the fit of the model to the data:
R2(S)=1−∣∣Y−Xβ^(S)∣∣2∣∣Y−Y‾n∣∣2R^2(S) = 1 - \frac{\norm{Y - X \hat{\beta}(S)}^2}{\norm{Y - \overline{Y}_n}^2}R2(S)=1−​​Y−Yn​​​2​​Y−Xβ^​(S)​​2​

  • The denominator is the sum of squared residuals for when we use the best constant function to fit
    • We use the constant E[Y]\exp{Y}E[Y]
  • If at least one of the variables in XXX for every instance is 111, then we can guarantee that this is at least 0
    • That is, we need to use linear regression with a constant term
    • This is because then it will do at least as well as the denominator
  • The larger the set SSS is, the larger this will be, so we will need a way to avoid overfitting
Forward / Greedy Model Selection

Initialize SSS to the empty set. Then, add the feature that causes the greatest increase in the R2R^2R2. Stop once you start to plateau or when you exceed some preset R2R^2R2 value
