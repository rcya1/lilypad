Block Multiplication

When multiplying AAA and BBB, can break them up into blocks

A=(A1,1…A1,n⋮Am,1…Am,n),B=(B1,1…B1,p⋮Bn,1…Bn,p)A = \begin{pmatrix}A_{1,1} & \dots & A_{1,n} \\ \vdots \\ A_{m,1} & \dots & A_{m,n}\end{pmatrix}, B = \begin{pmatrix}B_{1,1} & \dots & B_{1,p} \\ \vdots \\ B_{n,1} & \dots & B_{n,p}\end{pmatrix}A=​A1,1​⋮Am,1​​……​A1,n​Am,n​​​,B=​B1,1​⋮Bn,1​​……​B1,p​Bn,p​​​

Blocks don't have to be the same sizes, but they do have to be able to multiply with each other properly

Column Oriented Multiplication

In AB=CAB = CAB=C, each of the columns of CCC are a linear combination of the columns of AAA

  • Each column in BBB tells you the coefficients in that linear combination
  • Number of columns of AAA = number of rows of BBB

Row Oriented Multiplication

In AB=CAB = CAB=C, each of the columns of CCC are a linear combination of the rows of BBB

  • Each row in AAA tells you the coefficients in that linear combination
  • Number of columns of AAA = number of rows of BBB

Multiplying Diagonal Matrices

For a diagonal matrix DDD and matrix AAA:

  • DADADA multiplies the iii th row in AAA by the iii th element of DDD
  • ADADAD multiplies the iii th column in AAA by the iii th element of DDD
