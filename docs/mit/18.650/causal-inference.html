Causal Inference

A linear relationship between XXX and YYY does not imply causation

Causation implies:

  • Changing XXX necessarily changes YYY
  • Typically a time component, so the change in XXX has to occur before the change in YYY

Counterfactual Model

Let X∈{0,1}X \in \{0, 1\}X∈{0,1} be a random variable denoting whether or not a treatment was applied

  • X=0X = 0X=0 is the "control"
  • Y∈RY \in \realsY∈R is the response, which is typically a binary response but really could be any real

We define C0C_0C0​ and C1C_1C1​ to be the potential outcomes, i.e. the potential values of YYY in the case that X=0X = 0X=0 or X=1X = 1X=1

  • These are random variables
  • Y=CXY = C_XY=CX​
  • When X=0X = 0X=0, C0C_0C0​ is the observed, and C1C_1C1​ is the counterfactual. Vice versa for when X=1X = 1X=1

θ=E[C1]−E[C0]\theta = \exp{C_1} - \exp{C_0}θ=E[C1​]−E[C0​]

  • Generally, we do want to find the ATE, but it is not computable directly because we don't know the counterfactual for each observation

Estimating ATE with Association

  • We can design an experiment to guarantee θ=α\theta = \alphaθ=α
    • Known as randomized controlled trials, or A/B testing
    • This is done by determining who gets which treatment entirely by a random coin
      • This can be unethical, i.e. denying someone with a medical condition a treatment based on a flipped coin
  • As long as XXX is a pure Bernoulli random variable, then we guarantee that α=θ\alpha = \thetaα=θ
    • This is because C0C_0C0​ and C1C_1C1​ are independent of XXX
  • Often, we use observational studies where the outcomes are not independent of XXX
    • However, we might be able to find a feature vector ZZZ such that C0,C1C_0, C_1C0​,C1​ are independent of XXX conditioned on ZZZ
      • I.e. in a treatment, we could have ZZZ be a vector of all of your health conditions
    • We could then split up our observations based on the value of ZZZ, and for each of these, we would have that θ=α\theta = \alphaθ=α
  • However, this is hard to do when ZZZ is high-dimensional
    • You would have to create bins for each ZZZ, but then each bin would only have a small number of observations
    • We want to somehow reduce ZZZ to a single number, which is much easier to bin up
