Non Bayesian Parameter Estimation

In many problems, it is unnatural to assign a prior distribution to the latent variable of interest

  • We instead treat it as deterministic, but unknown
  • Instead of having observation models conditioned on some variable xxx, we instead have it be parameterized by xxx
    • This is the notation p(y∣x)p(y | x)p(y∣x) versus p(y;x)p(y; x)p(y;x)

The first thing to note is that if we look at our previous formulation of BLS:
x^=arg minfE[(x−f(y))2]\hat{x} = \t{arg min}_f \exp{(x - f(y))^2}x^=arg minf​E[(x−f(y))2]

  • However since xxx is now fixed, we have this is minimized when we just choose f(y)=xf(y) = xf(y)=x
  • We can't do this!
  • We need to restrict our search to valid estimators that are just statistics of the data

Estimators themselves are functions x^(y)\hat{x}(y)x^(y):

  • Their bias and variance are functions of xxx because we take the expectation over all possible yyy

For the beginning, we will focus on just scalar values of xxx (but yyy can be vector valued)

Minimum Variance Unbiased (MVU) Estimators

MVU

A MVU estimator x^∗\hat{x}_*x^∗​ is an estimator such that it has 0 bias and of all estimators that have 0 bias, it has the minimum variance λx^(x)\lambda_{\hat{x}}(x)λx^​(x) for every possible xxx

This could potentially not exist if:

  • There are no unbiased estimators
  • There are no estimators that uniformly have lower variance for all xxx

In general, there is no systematic procedure to find these or determine if they exist

Cramer-Rao Bound

Cramer-Rao Bound (Scalar Version)

Suppose py(y;⋅)p_y(y; \cdot)py​(y;⋅) for every yyy is positive and differentiable on X∈R\mc{X} \in \realsX∈R and satisfies the regularity condition for all xxx:
E[∂∂xln⁡py(y;x)]=0\exp{\pfrac{}{x} \ln p_y(y; x)} = 0E[∂x∂​lnpy​(y;x)]=0

Then for any unbiased x^(⋅)\hat{x}(\cdot)x^(⋅):
λx^(x)≥1Jy(x)\lambda_{\hat{x}}(x) \geq \frac{1}{J_y(x)}λx^​(x)≥Jy​(x)1​
where Jy(X)J_y(X)Jy​(X) is the Fisher information in yyy about xxx:
Jy(x)=E[S(y;x)2]J_y(x) = \exp{S(y;x)^2}Jy​(x)=E[S(y;x)2]
where S(y;x)S(y;x)S(y;x) is the score function for xxx based on yyy:
S(y;x)=∂∂xln⁡py(y;x)S(y;x) = \pfrac{}{x} \ln p_y(y;x)S(y;x)=∂x∂​lnpy​(y;x)

With an extra regularity condition, we can express the Fisher information as:

Corollary

If py(y;⋅)p_y(y; \cdot)py​(y;⋅) also satisfies the regularity condition:
E[(∂∂xln⁡py(y;x))2+∂2∂x2ln⁡py(y;x)]=0\exp{\p{\pfrac{}{x} \ln p_y(y;x)}^2 + \frac{\partial^2}{\partial x^2} \ln p_y(y;x)} = 0E[(∂x∂​lnpy​(y;x))2+∂x2∂2​lnpy​(y;x)]=0
then the Fisher information can also be expressed as:
Jy(x)=−E[∂2∂x2ln⁡py(y;x)]J_y(x) = -\exp{\frac{\partial^2}{\partial x^2} \ln p_y(y;x)}Jy​(x)=−E[∂x2∂2​lnpy​(y;x)]

  • The Fisher information cannot be computed in all problems
    • In these cases, no Cramer-Rao bound exists
  • The Fisher information can be interpreted as a measure of curvature
    • I.e. how "peaky" ln⁡py(y;x)\ln p_y(y; x)lnpy​(y;x) is as a function of xxxxxx
    • The larger Jy(x)J_y(x)Jy​(x), the better we expect to be able to resolve the value of xxx from observations
    • Therefore, the larger the Fisher information, the smaller we expect λx^(x)\lambda_{\hat{x}}(x)λx^​(x) to be
  • Any estimator that satisfies Cramer-Rao with equality must be a MVU estimator
    • The converse is not true, the Cramer-Rao bound could be not tight
  • There are generalizations for biased estimators

Efficiency

Efficient Unbiased Estimator

An unbiased estimator is efficient if it satisfies the Cramer-Rao bound with equality for all x∈Xx \in \mc{X}x∈X

Corollary

An estimator x^\hat{x}x^ is efficient iff it can be expressed in the form:
x^(y)=x+1Jy(x)S(y;x)\hat{x}(y) = x + \frac{1}{J_y(x)} S(y; x)x^(y)=x+Jy​(x)1​S(y;x)
where the right hand side must be independent of xxx for the estimator to be valid

  • An efficient estimator is guaranteed to be unbiased
  • An efficient estimator is unique
  • An efficient estimator must be the unique MVU estimator for a problem

Maximum Likelihood Estimation

MLE

The maximum likelihood estimate x^ML(y)\hat{x}_\text{ML}(y)x^ML​(y) for parameter xxx is:
x^ML(y)=arg max⁡x∈Xpy(y;x)\hat{x}_\text{ML}(y) = \argmax_{x \in \mc{X}} p_y(y; x)x^ML​(y)=x∈Xargmax​py​(y;x)

It is often convenient to use the likelihood function notation Ly(x)=py(y;x)L_y(x) = p_y(y; x)Ly​(x)=py​(y;x)

  • This emphasizes the perspective in which it is a dependence on xxx for a fixed yyy that is of interest
Corollary

Suppose \mc{X}$ is open, and that an efficient estimator and ML estimator both exist. Then the two are identical.

This does not mean that ML estimators are always efficient!

  • When an efficient estimator doesn't exist, then the ML estimator might not have special properties
    • I.e. they might not have good variance properties or even be unbiased

One nice property is that the ML estimation commutes with invertible mappings

  • If we have θ=g(x)\theta = g(x)θ=g(x), then θ^ML(y)=g(x^ML(y))\hat{\theta}_\text{ML}(y) = g(\hat{x}_\text{ML}(y))θ^ML​(y)=g(x^ML​(y))
  • This is not a property that is shared with other estimators (i.e. Bayesian estimators)
  • This doesn't apply when ggg isn't invertible, but it can be extended

Gauss-Markov Theorem

We look at a connection between least-squares and MLE estimation when looking at linear models with Gaussian data

Gauss-Markov Theorem

Suppose data yyy depends on xxx through the model:
y=Hx+wy = Hx + wy=Hx+w
with w∼N(0,Λw)w \sim \mc{N}(0, \Lambda_w)w∼N(0,Λw​) where Λw\Lambda_wΛw​ and HHH are full rank. Then the MLE estimator x^ML(y)\hat{x}_\t{ML}(y)x^ML​(y):
x^ML(y)=(HTΛw−1H)−1HTΛw−1y\hat{x}_\t{ML}(y) = \p{H^T\Lambda_w^{-1}H}^{-1} H^T \Lambda_w^{-1}yx^ML​(y)=(HTΛw−1​H)−1HTΛw−1​y
is the solution to a weighted least-squares problem and is a MVU.

The weighted least squares problem is where you find:
arg min⁡x(y−Hx)TΛw−1(y−Hx)\argmin_x (y-Hx)^T \Lambda_w^{-1} (y - Hx)xargmin​(y−Hx)TΛw−1​(y−Hx)

Corollary

When Λw∝I\Lambda_w \propto IΛw​∝I, the ML estimator is the solution to the ordinary least squares problem
x^ML(y)=arg min⁡x∣∣y−Hx∣∣22\hat{x}_\t{ML}(y) = \argmin_x \norm{y - Hx}_2^2x^ML​(y)=xargmin​∣∣y−Hx∣∣22​

Sketch:

  • Find the MLE by maximizing the likelihood, which turns into minimizing the exponential, which turns out to be exactly the least squares problem
  • Can prove that it is of the given form by taking the derivative
  • Can show it is a MVU by finding the Fisher information

In linear regression, we try to fit a relationship of the form:
