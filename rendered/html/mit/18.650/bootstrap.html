Bootstrap

So far, our confidence intervals have relied on asymptotic normality

  • We rely on being able to calculate asymptotic variance based on:

    • CLT if θ^=Xn‾\hat{\theta} = \overline{X_n}θ^=Xn​​

    • Fisher information if θ^=θ^MLE\hat{\theta} = \hat{\theta}^\text{MLE}θ^=θ^MLE

  • However, what if it is not in one of these three forms?

  • What if we tried to get an estimator for the mean of a Poisson variable?

    • We could estimate λ\lambdaλ with Xn‾\overline{X_n}Xn​​
    • θ\thetaθ is the integer such that:
      • ∑k=0θ−1p(k)<0.5\sum_{k=0}^{\theta-1} p(k) < 0.5∑k=0θ−1​p(k)<0.5

  • The key idea is that to estimate the variance of our estimator θ^\hat{\theta}θ^, we need multiple estimates

    • Imagine we had B⋅nB \cdot nB⋅n samples: then we could compute BBB estimators and then take their variance
Bootstrap Sample

Given nnn samples XiX_iXi​, a bootstrap sample is a collection of mmm samples Xi∗X_i^*Xi∗​ such that:
Xi∗∼P^n=Unif(X1,…,Xn)X_i^* \sim \hat{\mathbb{P}}_n = \text{Unif}(X_1, \dots, X_n)Xi∗​∼P^n​=Unif(X1​,…,Xn​)

This distribution is known as the empirical distribution of the data

  • The corresponding CDF is:
    F^n(x)=1n∑1(Xi≤x)\hat{F}_n(x) = \frac{1}{n} \sum 1(X_i \leq x)F^n​(x)=n1​∑1(Xi​≤x)
  • With the LLN, one can show this CDF converges in probability to F(x)F(x)F(x) as nnn goes to ∞\infty∞

Then, we can compute BBB bootstrap samples each of size nnn and can then use the sample variance vbootv_\text{boot}vboot​

  • Note there are two approximations:
    vboot≈VP^n[θ^]≈VP[θ^]v_\text{boot} \approx \mathbb{V}_{\hat{\mathbb{P}}_n}[\hat{\theta}] \approx \mathbb{V}_{\mathbb{P}}[\hat{\theta}]vboot​≈VP^n​​[θ^]≈VP​[θ^]
  • The first one can be made accurate by just increasing BBB
  • The second one relies on the number of samples nnn which we can't change
