Exponential Families

An exponential family is a parameterized family of distributions

Single-Parameter Exponential Family

For some X⊂RX \subset \realsX⊂R, a parameterized family of distributions over the alphabet YYY is a single parameter exponential family if it can be expressed in the form:
py(y;x)=exp(λ(x)t(y)−α(x)+βy)p_y(y; x) = \t{exp}\p{\lambda(x)t(y) - \alpha(x) + \beta{y}}py​(y;x)=exp(λ(x)t(y)−α(x)+βy)
for some choice of functions λ,t,β\lambda, t, \betaλ,t,β

  • λ\lambdaλ: natural parameter
  • ttt: natural statistic
  • β\betaβ: log base function

α\alphaα is not part of the specification because it is for normalization

  • A family may become no longer possible to be normalized for specific xxx

Z(x)=eα(x)=∑yexp(λ(x)t(y)+β(y))Z(x) = e^{\alpha(x)} = \sum_y \t{exp}\p{\lambda(x) t(y) + \beta(y)}Z(x)=eα(x)=y∑​exp(λ(x)t(y)+β(y))

  • Z(x)Z(x)Z(x) is known as the partition function
  • α(x)\alpha(x)α(x) is then known as the log-partition function

We denote that yyy follows an exponential family by:
y∼p∈E(X,Y,λ,t,β)y \sim p \in \mc{E}(\mc{X}, \mc{Y}, \lambda, t, \beta)y∼p∈E(X,Y,λ,t,β)

We note that if we instead factor out β(y)\beta(y)β(y) to get:
py(y;x)∝q(y)exp(λ(x)t(y))p_y(y; x) \propto q(y) \t{exp}\p{\lambda(x)t(y)}py​(y;x)∝q(y)exp(λ(x)t(y))

  • If we can get q(y)q(y)q(y) to be a distribution (integrate to $1$) then this is known as the base distribution of the family
Canonical Single-Parameter Exponential Family

A canonical exponential family is one with λ(x)=x\lambda(x) = xλ(x)=x

Natural Parameter Set

The natural parameter space is the collection of all possible xxx such that the corresponding form of the distribution can be normalized

  • This is a convex set, which we will see shortly
  • When Y\mc{Y}Y is finite, it is an open set
Natural Exponential Families

Canonical exponential families with Y⊂R\mc{Y} \subset \realsY⊂R and t(y)=yt(y) = yt(y)=y are referred to as natural exponential families

Exponential Family Constructions and Interpretations

Many familiar parameterized distributions can be expressed as exponential families:

  • Bernoulli random variable
  • Binomial random variable
  • Guassian random variable
  • Exponential random variable

The uniform random variable is not exponential
